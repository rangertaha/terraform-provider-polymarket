[![Test](https://github.com/rangertaha/terraform-provider-polymarket/actions/workflows/test.yml/badge.svg)](https://github.com/rangertaha/terraform-provider-polymarket/actions/workflows/test.yml)

A [Terraform](https://www.terraform.io) provider for [Polymarket](https://polymarket.com),
covering its three public APIs (Gamma catalog, CLOB order book, and Data/portfolio)
plus authenticated trading and on-chain approvals. Built on the modern
[terraform-plugin-framework](https://github.com/hashicorp/terraform-plugin-framework).

> Status: **feature-complete against the [coverage plan](./docs/API_COVERAGE_PLAN.md)** â€”
> 15 data sources and 3 resources. Read-only data sources work with no credentials;
> trading/allowance resources require a wallet `private_key` (and, for allowances,
> an `rpc_endpoint`). Live trading and on-chain approvals are best exercised on the
> Amoy testnet.

## Requirements

- [Terraform](https://developer.hashicorp.com/terraform/downloads) >= 1.0
- [Go](https://go.dev/dl/) >= 1.24 (to build the provider)

## Using the provider

```hcl
terraform {
  required_providers {
    polymarket = {
      source  = "rangertaha/polymarket"
      version = "~> 0.1"
    }
  }
}

provider "polymarket" {}

data "polymarket_markets" "active" {
  active = true
  closed = false
  limit  = 10
}

output "questions" {
  value = [for m in data.polymarket_markets.active.markets : m.question]
}
```

### Provider configuration

| Argument   | Env var               | Default                            | Description                                   |
| ---------- | --------------------- | ---------------------------------- | --------------------------------------------- |
| `endpoint` | `POLYMARKET_ENDPOINT` | `https://gamma-api.polymarket.com` | Base URL of the Gamma API.                    |
| `api_key`  | `POLYMARKET_API_KEY`  | _(none)_                           | Optional bearer token; not needed for public data. |

## Data sources

Read-only; no credentials required.

| | |
| --- | --- |
| **Gamma catalog** | `polymarket_market`, `polymarket_markets`, `polymarket_event`, `polymarket_events`, `polymarket_series`, `polymarket_tags` |
| **CLOB pricing** | `polymarket_order_book`, `polymarket_order_books` (batch), `polymarket_price`, `polymarket_prices` (batch), `polymarket_midpoint`, `polymarket_spread`, `polymarket_price_history` |
| **Data / portfolio** | `polymarket_positions`, `polymarket_trades`, `polymarket_portfolio_value`, `polymarket_holders` |
| **Auth** | `polymarket_api_credentials` (requires `private_key`) |

## Resources

Require a wallet `private_key`. `polymarket_allowance` also requires an `rpc_endpoint`.

- `polymarket_order` â€” place/cancel a CLOB limit order.
- `polymarket_api_key` â€” provision/revoke an L2 API key set.
- `polymarket_allowance` â€” manage on-chain ERC-20 / ERC-1155 trading approvals.

See [`examples/`](./examples) for runnable configurations and [`docs/`](./docs)
for the full per-attribute reference (generated by `tfplugindocs`).

## Developing

```sh
make build      # compile the provider binary
make test       # unit tests
make testacc    # acceptance tests (hit the live API)
make docs       # regenerate registry docs from schema descriptions
make fmt vet    # format and vet
```

To test locally, add a dev override to `~/.terraformrc`:

```hcl
provider_installation {
  dev_overrides {
    "rangertaha/polymarket" = "/path/to/your/GOBIN"
  }
  direct {}
}
```
